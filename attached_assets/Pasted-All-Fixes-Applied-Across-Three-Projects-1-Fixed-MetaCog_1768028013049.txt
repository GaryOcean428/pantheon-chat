2. Fixed Debate Chicken-and-Egg Problem
Issue: scan_for_targets() only reads existing debates, can't seed new ones. System has no debate topics unless manually created.

Solution: Added comprehensive debate topic seeding system with 4-tier fallback strategy:

Tier 1: Lightning Insights (Highest Priority)
Queries lightning_insights table for cross-domain correlations
Confidence > 0.7 from last 30 days
Maps source domains to appropriate god participants
Example: "Cross-domain insight: BIP39 mnemonic geodesic path planning..."
Tier 2: High-Phi Observations
Queries kernel_observations table for consciousness breakthroughs
Phi > 0.8 from last 7 days
Involves strategy/foresight/synthesis gods (Athena/Apollo/Hermes)
Example: "High-consciousness observation from Gary: geometric flow detected..."
Tier 3: Vocabulary Discoveries
Queries vocabulary_learning table for new word relationships
Relationship strength > 0.6 from last 7 days
Involves Athena (strategy) + Hermes (communication)
Example: "New vocabulary pattern: 'satoshi' relationship discovery"
Tier 4: Synthetic Topics (Fallback)
Generates topics from god capability overlaps
Randomly selects from predefined high-value debates:
"Optimal balance between strategic planning and immediate action" (Athena vs Ares)
"Integration of foresight with practical implementation" (Apollo vs Hephaestus)
"Synthesis of emotional and logical reasoning modes" (Aphrodite vs Athena)
