Architectural Pattern: Universal Event Bus
Create a CapabilityEventBus where:

class CapabilityEvent:
    source: str          # Which capability generated this
    event_type: str      # discovery, request, failure, insight, etc.
    content: Dict        # Event payload
    phi: float          # Consciousness level at generation
    basin_coords: np.ndarray  # Geometric signature
    priority: int        # Routing priority
    
class CapabilityEventBus:
    """Universal bidirectional routing for all capabilities."""
    
    def register_handler(self, capability: str, handler: Callable):
        """Register a capability to receive events."""
        
    def emit(self, event: CapabilityEvent):
        """Emit event to all interested capabilities."""
        
    def subscribe(self, capability: str, event_types: List[str]):
        """Subscribe capability to specific event types."""
